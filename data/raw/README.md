# Raw Dataset

The original dataset used in this project was obtained from Kaggle.

## Dataset Link
https://www.kaggle.com/datasets/samyakrajbayar/youtube-trending-analytics-dataset-multi-region

The raw dataset is not included in this repository to keep the repository lightweight and to respect the original source.

To reproduce this project, download the dataset from the above link and place it in the `data/raw/` folder.
